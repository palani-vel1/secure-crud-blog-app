# 🚀 Secure CRUD Blog Application

A modern and secure CRUD Blog Application built using **PHP**, **MySQL**, **HTML**, and **CSS** with authentication, role-based access control, validation, search functionality, pagination, and modern dashboard UI.

---

# 📌 Features

## 🔐 Authentication System

* User Registration
* Secure Login System
* Password Hashing
* Session Management
* Logout Functionality

---

## 🛡️ Security Features

* SQL Injection Protection using Prepared Statements
* Server-side Validation
* Client-side Validation
* Secure Password Verification
* Role-based Access Control

---

## 👤 User Roles

### Admin

* Add Posts
* Edit Posts
* Delete Posts
* Manage Blog Content

### User

* Add Posts
* Edit Posts
* Restricted Delete Access

---

## 📝 CRUD Operations

* Create Blog Posts
* Read Blog Posts
* Update Blog Posts
* Delete Blog Posts

---

## 🔍 Additional Features

* Search Functionality
* Pagination System
* Responsive Dashboard
* Modern Glassmorphism UI
* Sidebar Navigation

# ⚙️ Setup Instructions

## 1️⃣ Install XAMPP

Download and install XAMPP.

## 2️⃣ Move Project Folder

Place the project inside:

```plaintext
xampp/htdocs/
```

## 3️⃣ Start Apache & MySQL

Open XAMPP Control Panel and start:

* Apache
* MySQL

## 4️⃣ Create Database

Open:

```plaintext
http://localhost/phpmyadmin
```

Create database:

```plaintext
blog
```

Create required tables:

* users
* posts

---

# ▶️ Run Project

Open browser:

```plaintext
http://localhost/crud_app
```
