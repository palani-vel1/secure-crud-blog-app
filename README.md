# Modern CRUD Blog Application

A modern CRUD Blog Application built using PHP and MySQL with authentication, search, pagination, and dark dashboard UI.

## Features

- User Registration & Login
- Create Post
- Read Posts
- Update Posts
- Delete Posts
- Pagination
- Search Functionality
- Modern Dark Dashboard UI
- Responsive Design

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP
